# Finance Stress Simulator - Frontend

Beautiful React interface for the Financial Stress Simulator.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start
```

The app will open at http://localhost:3000

**Make sure the backend is running on port 8000!**

## What You'll See

1. **Input Form** - Enter your financial information
2. **Results Dashboard** - See 6 different "what if" scenarios with:
   - Visual charts showing your balance over time
   - Runway calculations (how many months until broke)
   - Risk levels (high/medium/low)
   - Actionable recommendations with quantified impact
